    Welcome to Syncify!
    Here is a little introduction about this project.
    Syncify is an Application created by FindMalek, since 2020 April.
    Using this app, you can download your Spotify playlist and listen to it offline.
    For more questions about how the Application works, contact me at:
    Malekgarahellalbus@gmail.com
    

    You can use any of these commands (Simply enter the number of the command):
    1. Enter Playlist / Albums / Tracks links 
        Press <Enter> to stop.
    2. Start downloading the saved Playlists / Albums / Tracks.
        Downloads all the Playlists / Albums / Tracks that are saved, and make a playlist file <.m3u>.
    3. Check the saved Playlists / Albums / Tracks.
        Print all the saved Playlists / Albums / Tracks
    4. Change the setting of Syncify.
        |
        -> 1. Change the download quality.
                Recommended to keep it <BEST>
           2. Change the download format.
                Recommended to keep it <MP3>
           3. Change the location of the downloaded tracks.
                Where Syncify downloads the tracks.
           4. Change the order of downloading (By default it's Playlists -> Albums -> Tracks).
                How Syncify prioritize downloading.           
           5. Change the location of the stored playlist (.m3u)
                Where the playlist <.m3u> files are stored.
    5. Correct the track.
        Use this command when you download a track but it dosen't match the version in the Spotify.
        <This command is coming in the next updates...>
    6. Exit.
